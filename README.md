# 📚 Library Management System

## 📌 Project Overview
This is a desktop-based Library Management System built using Java (Swing) and MySQL.
It allows managing books, members, transactions, and reports efficiently.

## 🎯 Features
- User Login System 🔐
- Book Management (Add, Update, Delete, Search)
- Member Management 👥
- Borrow & Return System 🔄
- Auto Overdue Detection ⚠️
- Reports Dashboard 📊
- Export Reports to PDF & Excel

## 🛠️ Technologies Used
- Java (Swing)
- MySQL
- JDBC
- iText (PDF export)

## ⚙️ Setup Instructions
1. Install Java (JDK 17+)
2. Install MySQL
3. Run `database/schema.sql`
4. Add MySQL Connector + iText JAR
5. Run Main.java

## ▶️ How to Run
- Open project in VS Code
- Run `Main.java`
- Login with:
  - Username: admin
