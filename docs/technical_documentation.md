# Technical Documentation

## Architecture
- MVC Pattern
  - Model → Book, Member, Transaction
  - DAO → Database operations
  - GUI → Swing panels

## Database Design
- Books Table
- Members Table
- Transactions Table
- Users Table

## Key Features
- JDBC connection
- PreparedStatement usage
- Trigger-based automation

## Algorithms
- Search using LIKE query
- Overdue detection using date comparison
